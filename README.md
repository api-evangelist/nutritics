# Nutritics (nutritics)

Nutritics provides a real-time food and nutrition data API offering access to over one million foods (generic, branded, and recipes), up to 258 nutrient parameters, allergen data, and regional compliance options across the EU, US, Australia, New Zealand, and South Africa.

## APIs

### Nutritics Food Data API
REST API providing access to over one million foods including generic, branded, and recipe items, with up to 258 nutrient parameters, allergen information, and regional compliance datasets. JSON responses, basic authentication, GET and POST methods.
